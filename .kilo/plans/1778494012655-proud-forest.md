**Nuova tabella proposta:**
```python
class MappingRule(Base, UUIDMixin, TimestampMixin):
    __tablename__ = "mapping_rules"
    hotel_id: UUID (FK, NOT NULL)  # ogni hotel ha le sue regole
    entity_type: MappingType enum  # centro_costo, attività, servizio, conto
    external_code_pattern: str      # regex o wildcard pattern (es. "PMS_*", "CC-001")
    internal_entity_id: UUID        # FK a entità target (cost_center, activity…)
    match_type: str                 # "exact" | "regex" | "fuzzy" | "range"
    priority: int = 0               # priorità regola (più alta = prima)
    active: bool = True
    description: Optional[str]
```

**Endpoint API:**
- `GET /mapping-rules` — lista con filtri hotel/entity_type
- `POST /mapping-rules` — crea regola (validate regex se match_type=regex)
- `PUT /mapping-rules/{id}` — modifica
- `DELETE /mapping-rules/{id}` — elimina
- `POST /mapping-rules/apply-template` — applica template predefinito per PMS tipo
- `GET /mapping-rules/test` — testa regola su sample data (senza salvare)

**UI:**
- Nuova pagina `/mapping-rules` con tabella editabile
- Form con: entity_type selector, external pattern (con help tipo wildcard es. `PMS_*`), search dropdown per entità interna

### **PATCH BATCH 1: Core Dependencies & Auth (Settimana 2)**

**Task 1.1 — Create `backend/app/core/dependencies.py`** (NUOVO FILE)

```python
"""
Tenant-aware dependencies for multi-tenancy enforcement.
"""
from fastapi import Depends, HTTPException, Header, Request
from sqlalchemy.ext.asyncio import AsyncSession
from app.db.database import get_db
from app.models.models import User, Hotel
from app.services.user_service import UserService  # se esiste, altrimenti creare

async def get_current_hotel(
    request: Request,
    db: AsyncSession = Depends(get_db),
) -> str | None:
    """
    Estrae hotel_id da:
    1. Header X-Hotel-ID (per admin multi-hotel)
    2. JWT token claim (per utente normale)
    3. Query param ?hotel_id= (fallback)
    """
    # Try header first (admin switching)
    x_hotel_id = request.headers.get("X-Hotel-ID")
    if x_hotel_id:
        return x_hotel_id
    
    # Fallback: from user's default hotel (will be set by auth dependency)
    # This return None if user has no hotel_id (which is fine for super-admin)
    return None


def require_hotel_access(hotel_id: str | None = Depends(get_current_hotel)):
    """
    Dependency that ensures the current user has access to the specified hotel.
    Use this on endpoints that operate on hotel-scoped resources.
    """
    if not hotel_id:
        raise HTTPException(
            status_code=403, 
            detail="Hotel context required. Provide X-Hotel-ID header."
        )
    return hotel_id
```

```python
from fastapi import Request
import json

class AuditLogMiddleware:
    async def __call__(self, request: Request, call_next):
        response = await call_next(request)
        
        # Log solo operazioni scrittura (POST/PUT/DELETE)
        if request.method in ("POST", "PUT", "DELETE", "PATCH"):
            user = request.state.user  # da auth
            hotel_id = request.headers.get("X-Hotel-ID", "global")
            log_entry = {
                "timestamp": datetime.utcnow().isoformat(),
                "user_id": user.id if user else None,
                "hotel_id": hotel_id,
                "method": request.method,
                "path": request.url.path,
                "client_ip": request.client.host,
            }
            logger.info(f"AUDIT: {json.dumps(log_entry)}")
        
        return response
```
